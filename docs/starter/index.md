# 快速开始

## 安装

### EchartsLayer

::: code-group

```sh [pnpm]
$ pnpm add @naivemap/maplibre-gl-echarts-layer
```

```sh [npm]
$ npm add @naivemap/maplibre-gl-echarts-layer
```

```sh [yarn]
$ yarn add @naivemap/maplibre-gl-echarts-layer
```

:::
